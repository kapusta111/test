показывает статус изменений 
git status

перед тем как отправить изменения на сервер надо определить какие файлы добавить в него
git add filename_подставь_свое_название_файла

делаем локальный (у себя на компе) комит (сохраниения кода)
git commit -m 'тут_твое_сообщение'

отправляем созданный коммит на сервер (в ту ветку в которой ты работаешь)
git push
